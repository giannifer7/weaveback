= Weaveback CLI Macros
:toc: left

Shared macro definitions imported by `cli-spec.adoc` and each standalone
binary CLI spec.  Not a driver file — contains only `%def` blocks that
produce no output themselves.

== Shared renderers

[source]
----
%def(render_rust_struct_option, field, rust_type, arg_attr, doc_block, doc_flag, doc_default, doc_descr, %{%(doc_block)
%if(%(arg_attr), %{    #[arg(%(arg_attr))]
%})
    pub(crate) %(field): %(rust_type),
%})

%def(render_rust_enum_option, field, rust_type, arg_attr, doc_block, doc_flag, doc_default, doc_descr, %{%(doc_block)
%if(%(arg_attr), %{    #[arg(%(arg_attr))]
%})
    %(field): %(rust_type),
%})

%def(render_doc_option, field, rust_type, arg_attr, doc_block, doc_flag, doc_default, doc_descr, %{| %(doc_flag) | %(doc_default) | %(doc_descr)
%})
