= wb-mcp CLI spec

%import(cli-spec/macros.adoc)

This file defines the standalone `wb-mcp` CLI.

The hand-written `db` / `gen_dir` fields stay inline because they are specific
to this binary. The reusable evaluator flags are projected into a named chunk.

== Rust projection chunk

[source]
----
%alias(emit_option, render_rust_struct_chunk_option)
%eval_options(chunk_name = wb-mcp-eval-options)
----

[source,rust]
----
// <<@file crates/wb-mcp/src/cli_generated.rs>>=
use clap::Parser;
use std::path::PathBuf;

/// Weaveback MCP server: JSON-RPC bridge for AI agent tooling.
///
/// Reads JSON-RPC 2.0 requests from stdin, writes responses to stdout.
/// Intended for use as a stdio-based MCP server.
#[derive(Parser, Debug)]
#[command(name = "wb-mcp", version)]
pub(crate) struct Cli {
    /// Path to the weaveback database.
    #[arg(long, default_value = "weaveback.db")]
    pub(crate) db: PathBuf,

    /// Base directory for generated output files.
    #[arg(long = "gen", default_value = "gen")]
    pub(crate) gen_dir: PathBuf,

// <<wb-mcp-eval-options>>
}
// @
----
