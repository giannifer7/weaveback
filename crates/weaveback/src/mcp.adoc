= MCP Server (re-export)
:toc: left
:source-highlighter: syntect

Re-exports `weaveback-api::mcp` so the combined `weaveback` binary
can continue calling `mcp::run_mcp` without call-site changes.

[source,rust]
----
// <<mcp-reexport>>=
pub use weaveback_api::mcp::*;
// @
----

----
