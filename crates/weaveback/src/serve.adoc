= weaveback serve (re-export)
:toc: left
:source-highlighter: syntect

Re-exports the `weaveback-serve` library crate so that the combined `weaveback`
binary can continue using `serve::run_serve`, `serve::AiBackend`, and
`serve::TangleConfig` without any call-site changes.

[source,rust]
----
// <<serve-reexport>>=
pub use weaveback_serve::*;
// @
----

----
// <<@file weaveback/src/serve.rs>>=
// <<serve-reexport>>
// @
----
